📊 Telecom X – Análise de Evasão de Clientes (Churn de Clientes) - Parte 2

Este projeto analisa o comportamento de clientes de uma empresa de telecom para prever a probabilidade de cancelamento dos serviços (churn).  
Foram treinados modelos de Machine Learning, como Regressão Logística e Random Forest, para identificar clientes com maior risco de evasão.  
O objetivo é fornecer insights que ajudem a empresa a implementar estratégias de retenção direcionadas.
